Demo of various Pipes

Demo1: camelCaseToWords pipe

It converts a string in camel case to string of words separated by spaces.

original string => {{ camelCaseString }}
string after pipe => {{ camelCaseString | camelCaseToWords }}

Demo2: replaceCharacter pipe

It is used to replace a character with any other character.

  • @param (string) value
  • @param (string) [toReplace='_'] the character to be replaced // default replaces '_' (underscore)
  • @param (string) [replaceWith=' '] the character to be replaced with // by-default it is replaced with blank space
original string => {{ replaceCharString }}
string after pipe (replaceCharString | replaceCharacter) => {{ replaceCharString | replaceCharacter }}
string after pipe (replaceCharString | replaceCharacter: "_":"~*~") => {{ replaceCharString | replaceCharacter: "_":"~*~" }}
