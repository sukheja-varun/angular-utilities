Demo of Date Extension

We have added a file (date.extension.ts) to this project, which uses prototypal inheritance to add new functionalities to JS Date object.

Demo1: clone()

Here we will be using the new clone() method to clone an existing date object.

Demo2: setStartHours()

It mutates the date object and sets the time (HH:mm:ss.SSS) to start time of the day i.e. 00:00:00.000

Demo3: setEndHours()

It mutates the date object and sets the time (HH:mm:ss.SSS) to end time of the day i.e. 23:59:59.999

Demo4: addDays()

It adds the number of days given as function argument to the date object and returns the new date formed

Demo5: isToday()

It checks if calendar date(DD:MM:YYYY) of current date object is today date

Demo6: compareDate()

It compare date object (date and time) with the date in function argument.

  • returns -1 if currentDate (date and time) is smaller then argument date object
  • returns 0 if currentDate (date and time) equals argument date object
  • returns 1 if currentDate (date and time) is larger then argument date object

Demo7: compareCalendar()

It compare date object (DD:MM:YYYY) with the date in function argument.

  • returns -1 if currentDate (DD:MM:YYYY) is smaller then argument date object
  • returns 0 if currentDate (DD:MM:YYYY) equals argument date object
  • returns 1 if currentDate (DD:MM:YYYY) is larger then argument date object
